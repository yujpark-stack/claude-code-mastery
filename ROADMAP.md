# 개발자 웹 이력서 개발 로드맵

## 📋 프로젝트 개요
개인의 경력과 기술을 보여주는 반응형 웹 이력서 개발

### 목표
- 전문적이고 깔끔한 이력서 포트폴리오 구현
- 반응형 디자인으로 모든 기기에서 완벽하게 표시
- 빠른 로딩 및 최적화된 성능

---

## 🛠️ 기술 스택
- **HTML5** - 마크업 및 구조
- **CSS3** - 스타일링 (Tailwind CSS 활용)
- **Tailwind CSS** - 유틸리티 기반 CSS 프레임워크
- **JavaScript** - 인터랙션 및 동적 기능
- **Git** - 버전 관리

---

## 📁 프로젝트 구조

```
web-resume/
├── index.html          # 메인 이력서 페이지
├── css/
│   └── styles.css      # Tailwind CSS 커스텀 스타일
├── js/
│   └── main.js         # 인터랙티브 기능
├── assets/
│   ├── images/         # 프로필 이미지 등
│   └── icons/          # 소셜 미디어 아이콘
├── tailwind.config.js  # Tailwind 설정
└── package.json        # 프로젝트 메타데이터
```

## 📝 이력서 콘텐츠 구성

### 기본 정보
- 이름: 김개발 (Generic name)
- 직급: 주니어 프론트엔드 개발자
- 이메일: dev@example.com
- 깃허브: github.com/example
- 링크드인: linkedin.com/in/example

### 경력 (경력사항)
```
1. 회사명 A - 프론트엔드 개발자 (2023.01 - 현재)
   - React, Vue를 활용한 웹 애플리케이션 개발
   - 컴포넌트 기반 UI 설계 및 구현

2. 회사명 B - 웹 개발 인턴 (2022.06 - 2022.12)
   - HTML/CSS/JavaScript를 활용한 웹사이트 개발
   - 디버깅 및 코드 리뷰 참여
```

### 기술 스택
- **Frontend**: HTML5, CSS3, JavaScript ES6+, React, Vue.js, Tailwind CSS
- **Tools**: Git, Webpack, npm, Visual Studio Code
- **Others**: REST API, Responsive Design

### 프로젝트
```
1. 개인 포트폴리오 웹사이트
   - HTML, CSS, JavaScript, Tailwind CSS 사용
   - 반응형 디자인 구현

2. Todo 애플리케이션
   - React로 만든 동적 할일 관리 앱
   - localStorage를 활용한 데이터 저장
```

### 학력
```
- OOO 대학교 컴퓨터공학 전공 (2020.03 - 2024.02)
```

### 자격증 및 언어
- TOEIC 900점
- 정보처리기사

## 🎨 디자인 가이드

### 색상 팔레트
- **Primary**: #2563eb (파란색)
- **Secondary**: #10b981 (초록색)
- **Dark**: #1f2937 (진회색)
- **Light**: #f3f4f6 (밝은 회색)
- **Accent**: #f59e0b (주황색)

### 타이포그래피
- **Heading**: 부드럽고 현대적인 폰트 (Poppins, Inter)
- **Body**: 가독성 높은 폰트 (Roboto, Open Sans)

### 레이아웃
- 모바일 우선 접근법
- 최대 너비: 1200px
- 여백과 정렬을 통한 깔끔한 구조
